OpenRecV2

The OpenRecV2 server (GS.NET) is being hosted on a Google site solely for the purpose of emulating Rec Room's network and providing an alternative experience for users. While it may not be as reliable or stable as an official server, we are working hard to ensure that it functions as smoothly as possible. We also understand the importance of user data security and have taken measures to protect it. Thank you for your understanding and for choosing OpenRecV2.

GS.NET provides APIs for accessing various types of data related to Rec Room, such as account information, relationships, rooms, inventions, and more.

GS.NET allows users to log in and authenticate themselves using its Auth API.

GS.NET hosts images that are shared within the Rec Room community.

GS.NET provides a CDN for content distribution.

GS.NET supports various features within Rec Room, such as chat, clubs, commerce, notifications, and matchmaking.

About OpenRecV2 OpenRecV2 is a modified version of the original OpenRec program created by Recroom 2016, which allows users to create custom servers for the 2016, 2017, and 2018 versions of Rec Room, all in one program.
